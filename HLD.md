## 2. Structural Architecture: The 8-Agent Neural Council
Instead of a single "Black Box" AI, we utilize a **Decoupled Workflow** consisting of eight specialized LLM nodes. This mimics a clinical board of experts, ensuring no domain-overlap and maximum logic-fidelity.

### Phase 01: Signal Acquisition (The Input Layer)
*   **Vitals Stream:** Normalized biometric data (HR, SpO2, Temp).
*   **Subjective Registry:** Unstructured symptom reports.
*   **Clinical Vault:** OCR-processed lab reports and imaging metadata.

### Phase 02: The Council (The Processing Layer)
1.  **LLM1 (The Orchestrator):** Analyzes intent and shards the query.
2.  **LLM2-7 (Specialist Nodes):** Individual agents for Symptoms, Nutrition, Mental Stability, and Lab Interpretation.
3.  **LLM8 (The Combiner):** De-duplicates specialist outputs and synthesizes a singular, verified protocol.

## 3. Data Integrity: Zero-Knowledge Enclave
Trust is not built on promises; it is built on **Architecture.**

*   **Identity Decoupling:** PII (Personally Identifiable Information) is hashed and stored in a separate enclave from biological data.
*   **RAG-8 Verified:** No agent is permitted to "generate" medical facts. Every output must be anchored to a vector-snippet from Tier-1 medical repositories (CDC, WHO, NHS).
*   **Deterministic Override:** A hard-coded logic layer (The Risk Engine) sits *above* the AI. If `Symptom = ACUTE_CHEST_PAIN`, the AI council is bypassed entirely for a millisecond-latency SOS escalation.
